## 买完怎么用？SSH 连接教程

服务器到手之后，很多人第一个问题是：怎么连？

邮件里会有三样东西：IP 地址、SSH 端口、root 密码。

Windows 用户用 PuTTY 或者 Xshell 连接，输入 IP 地址、端口、用户名 root 和密码就行。Mac/Linux 用户直接开终端，命令格式：

bash
ssh root@你的IP地址 -p 端口号


输入密码的时候不显示字符，这是正常的，盲打完按回车。第一次登录建议先改 root 密码（passwd 命令），然后可以开始折腾了。
